Design System — Tabs

3 modelos padrão para todas as páginas do dashboard

Modelo 1

FilterTabs

Busca + filtros. Usado em: Enquetes, Comunidades, Equipe, Proposições

<FilterTabs
  searchValue={search}
  onSearchChange={setSearch}
  tabs={[
    { value: 'todas', label: 'Todas' },
    { value: 'publicadas', label: 'Publicadas', count: 2 },
    { value: 'encerradas', label: 'Encerradas' },
    { value: 'arquivadas', label: 'Arquivadas' }
  ]}
/>
Modelo 2

ContentTabs

Só tabs, sem busca. Conteúdo muda por aba. Usado em: Comunicação, Relatórios, Documentos

Conteúdo da aba "Comunicados" aparece aqui

<ContentTabs
  tabs={[
    { value: 'comunicados', label: 'Comunicados', content: <ListaCampanhas /> },
    { value: 'templates', label: 'Templates', content: <ListaTemplates /> }
  ]}
/>
Modelo 3

IconTabs

Ícone + texto. Toggle visual. Usado em: Tarefas (Kanban/Lista), Eleitores (Mapa/Lista)

<IconTabs
  tabs={[
    { value: 'kanban', label: 'Kanban', icon: LayoutGrid },
    { value: 'lista', label: 'Lista', icon: List }
  ]}
  onChange={setView}
/>

Mapeamento por Página

Página Modelo Tabs
EnquetesFilterTabsTodas, Publicadas, Encerradas, Arquivadas
ComunidadesFilterTabsTodas, Com líder, Sem líder
EquipeFilterTabsTodos, Ativos, Inativos, Pendentes
ProposiçõesFilterTabsTodas, Em tramitação, Aprovadas, Rejeitadas
TarefasFilterTabs + IconTabsTodas, Pendentes, Em Andamento, Concluídas + Kanban/Lista
ComunicaçãoContentTabsComunicados, Templates
RelatóriosContentTabsGeral, Eleitores, Comunidades, Atendimentos
DocumentosContentTabsTodos, PDFs, Imagens, Planilhas
AgendaFilterTabsTodos, Reuniões, Eventos

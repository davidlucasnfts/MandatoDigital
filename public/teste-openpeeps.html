🧍 Tipos de Pose

O Open Peeps oferece três formatos de ilustração que podem ser usados em diferentes contextos.

Bust / Meia Figura

Apenas cabeça e ombros. Ideal para avatares, perfis e ícones pequenos.

Standing / Em Pé

Corpo inteiro em pé. Ideal para cenas, fluxos e ilustrações completas.

Sitting / Sentado

Corpo inteiro sentado. Ideal para cenas de escritório, reuniões, etc.

📥 Como Usar no Projeto

1

Acesse o Open Peeps

Vá para openpeeps.com ou use o plugin do Figma/Sketch

2

Monte seu Personagem

Escolha pose + expressão + roupa + cabelo + acessórios

3

Exporte em SVG

Baixe no formato SVG para poder mudar as cores via CSS

4

Integre ao Projeto

Use como imagem ou inline SVG nos clusters do mapa
